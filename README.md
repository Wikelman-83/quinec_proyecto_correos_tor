# Proyecto Correos Tor

Este proyecto automatiza la creación y gestión de cuentas, verificando IPs y ocultando identidad mediante la red Tor.

## Tecnologías utilizadas

- Python 3.10+
- Selenium
- Tor Expert Bundle
- ChromeDriver
- SQLite
